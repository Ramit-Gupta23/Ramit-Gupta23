## Projects

### [TerraStorm – Cloud-Native Land Monitoring](https://github.com/Ramit-Gupta23/TerraStorm-Cloud-Native-Land-Monitoring)

Event-driven AWS platform that detects land-use changes from satellite imagery — deforestation, urban expansion, water body shifts, illegal encroachment. Satellite images uploaded to S3 trigger Lambda, which invokes a CNN change-detection model on SageMaker. Results are stored back to S3, visualized through a Django dashboard, and critical alerts dispatched via Amazon SNS.

`Django` · `CNN (Siamese/Change Detection)` · `AWS SageMaker` · `Lambda` · `S3` · `SNS` · `Docker`

---

### [IMDB Sentiment Analysis – SimpleRNN](https://github.com/Ramit-Gupta23/IMDB_RNN)

End-to-end sentiment classifier on the Keras IMDB dataset. Covers the full cycle: preprocessing, sequence modeling with SimpleRNN, training stability analysis (exploding loss from ReLU vs. tanh, vanishing gradients), model serialization, and deployment via Streamlit. Documents architectural trade-offs of SimpleRNN vs. LSTM/GRU with concrete observations on overfitting and vocabulary constraints.

`TensorFlow` · `Keras` · `Streamlit` · `Jupyter`

---

### [Transformer from Scratch – GPT](https://github.com/Ramit-Gupta23/Transformer-from-Scratch)

Ground-up GPT implementation in a Jupyter notebook. Starts with a character-level Bigram Language Model as a baseline, then builds toward self-attention and multi-head attention blocks. Includes tokenization exploration with OpenAI's [tiktoken](https://github.com/openai/tiktoken) and [SentencePiece](https://github.com/google/sentencepiece).

`PyTorch` · `tiktoken` · `SentencePiece` · `Jupyter`

---

### [Spam Detection](https://github.com/Ramit-Gupta23/Spam_Detection)

Text classification pipeline using TF-IDF vectorization and a Naive Bayes classifier. The fitted vectorizer and model are serialized separately — keeping preprocessing and inference reproducible across environments. Deployed live on [Render](https://spam-detection-d1cu.onrender.com/).

`Scikit-Learn` · `TF-IDF` · `Pickle` · `Render`
